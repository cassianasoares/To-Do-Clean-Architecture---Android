# System Requirements
- Navigation Componet
- Data Binding
- ROOM persistence library
- Coroutines
- LiveData
- ViewModel

# Features

- Save, update and delete one item or all items
- Search item
- Sort by High or Low priority
- Swipe item to delete
- Undo deletion after swiping
- Change app to Dark or Light Theme
